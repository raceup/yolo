Requirements
  1. Connect the Kvaser and this laptop to the same Wi-Fi network.
  2. Find the IP of the Kvaser. It should be something like 192.168.1.12
  3. Set the same IP into the Settings window
  4. Select which logs to produce. The output will be available in the logs/ directory
  5. The software will connect to the Kvaser and start receiving CAN messages via the Canking REST API


For more info, please open a browser to https://github.com/raceup/yolo
